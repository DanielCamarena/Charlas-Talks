Información ambiental¶

Los datos ambientales usualmente involucran observaciones de variables ambientales en el tiempo y en el espacio, como cuando se quiere estudiar si la temperatura superficial terrestre se incrementar con el cambio climtico.

Tabla de escalas de resolucin espacial y temporal

Trabajando con datos ambientales¶

Por ello, es natural hablar de tablas o arreglos de datos, asi como de dimensiones espaciales y temporales.

Un estndar para el almacenamiento de datos ambientales es conocido como NetCDF. La principal ventaja de trabajar con el estndar netcdf es que ha sido optimizado para permitir una manipulacin prctica de los datos usando los metadatos asociados.

Hay diversas herramientas computacionales para realizar diagramas de Taylor: NCL, Python, Matlab.

Modelos climáticos¶

Un modelo climtico es un modelo numrico que se usa para reproducir el sistema climtico que incluyendo la atmsfera, el ocano, la superficie terrestre y el hielo.

Diagrama de Taylor¶

Sea $(\Omega, \mathcal{F}, P)$ un espacio de probabilidad, una funcin medible $X: \Omega \rightarrow \mathbb{R}$ se llama variable aleatoria.

Sobre el espacio de las variables aleatorias se pueden definir:

  • Funcin de distribucin de probabilidad $$ F_X(x)= P(X \le x)$$
  • Esperanza $$E[X]=\int X \, dP$$
  • Varianza $$\mathrm{var}(X)=E\left[ {(X-E[X])}^2 \right]$$ y la respectiva desviacin estndar $$\mathrm{sd}(X)=\sqrt{\mathrm{var}(X)}$$
  • Covarianza $$\mathrm{cov}(X,Y)=E\left[ (X-E[X])(Y-E[Y]) \right]$$ y el respectivo coeficiente de correlacin lineal $$\mathrm{cor}(X,Y)=\frac{\mathrm{cor}(X,Y)}{\mathrm{sd}(X)\mathrm{sd}(Y)}$$

Resultado: Se cumple que $$ \mathrm{var}(X-Y) = \mathrm{var}(X)-2\mathrm{cov}(X,Y)+\mathrm{var}(Y) $$ que puede escribirse como $$ E \left[ {\left((X-E[X])-(Y-E[Y])\right)}^2\right] = \mathrm{sd}(X)^2-2\mathrm{sd}(X)\mathrm{sd}(Y)\mathrm{cor}(X,Y)+\mathrm{sd}(Y)^2 $$ ms an

Es decir si entendemos $\mathrm{var}(X-Y)$ como el error cuadrtico medio centrado ($E'$) tenemos que este depende solamente del cociente de desviaciones estndar y la correlacin. $$ E'=\mathrm{var}(X-Y) = {\sigma_X}^2-2\sigma_X\sigma_Y\cos \left( \cos^{-1} \left(\mathrm{cor}(X,Y) \right) \right)+\sigma_Y^2 $$

Esto es el fundamento del Diagrama de Taylor que permite visualizar bidimensionalmente 3 mtricas que resumen las caractersticas de un modelo respecto una referencia. Posteriormente Glecker et al. propusieron el uso de diagramas de Taylor multivariados junto con dos ndices para evaluar la habilidad climtica y la variabilidad del modelo.

Comentarios adicionales:

  • Compararar toda la distribucin de probabilidad
  • Realizar filtros temporales y/o espaciales
  • Reto: una mtrica que mida si un modelo va a replicar la variabilidad a cierta escala temporal (alta frecuencia).

Caso de estudio: modelos atmosféricos globales y el mar peruano¶

Nosostros en especfico estamos interesados en evaluar el rendimiento de los modelos climticos del proyecto CMIP6 sobre el Sistema de Corrientes de Humboldt.

Sistema de afloramiento peruano¶

Primero una breve descripcin de los datos: El rea de estudio es el sistema de afloramiento peruano que es la parte norte del Sistema de la Corriente de Humboldt, uno de los cuatro principales ecosistemas de borde oriental de que contribuyen a la produccin pesquera mundial.

  • Dominio espacial: Lon 70 a 100 W, lat 0 a 25 S, z~10m sobre el nivel del mar
  • Dominio temporal: 01/01/1990 a 31/12/2010

Las variables de estudio son las variables atmsfericas

  • temperatura superficial (ts,tas)
  • nivel de presin atmosfrica (psl)
  • velocidad zonal (uas)
  • velocidad meridional (vas)

En el presente trabajo se van a manejar datos provenientes de dos fuentes de datos diarios

  • Reanlisis CFSR
  • Modelos climticos CMIP6: IPSL-CM6A-LR, MPI-ESM1-2-HR, MPI-ESM1-2-LR

    ts_stats['ccoef'].append(ccoef)

file2read.close()
